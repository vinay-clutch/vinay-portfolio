# Vinay Portfolio

Personal portfolio built with Next.js 16 and TypeScript.

## Features
- Modern responsive design
- Project showcase with live links
- Smooth animations using Framer Motion
- Optimized for performance and SEO

## Tech Stack
Next.js 16  
React 19  
TypeScript  
Tailwind CSS  
Framer Motion  
Vercel
